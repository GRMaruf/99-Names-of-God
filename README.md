# 99-Names-of-God
The Analysis of Allah's Beautiful Names.

### Important Note

Important Note on the Number of Names: Islamic theology holds that the names of Allah are not strictly limited to this number. The number 99 comes from the famous hadith which promises Paradise to those who 'count' (or learn, memorize, and act upon) them. There are many other names and attributes mentioned throughout the Quran and the Sunnah, and many are known only to Allah in the 'unseen.'
